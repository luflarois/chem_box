# chem_box
Chemistry standalone for BRAMS-MONAN

## how to compile
First, install fpm: see how to install in https://fpm.fortran-lang.org/

and use the command:

```
fpm install --compiler gfortran --flag "-O3"
```

*OBS: The binary will be installed in .local/bin in Your home area*

## Getting test data

Get the test set in https://drive.google.com/file/d/1ibYnNqpMlNfKEmhXsBnpKBsWm9CPD7Hr/view?usp=sharing
The data was generated by brams model, for 3 times, 0sec, 28800sec and 43200sec. Using 4 processors (1,2,3 and 4).
For test the BRAMS was executed using the Amazon area, 60 km of resolution.
The test set has a size of 2.1 GB

Download the data in a work diretory You prefer and unpack the data:

```
tar -xzvf test_data.tgz
```

## How to run

After unpack the data just run the box-model:

```
<your_home>>.local/bin/rodas3
```

When the program end, a set of files (csv) with chem species name will be created. The data will be displayed in each file, with the results os chem solver (concentration of the specie), and comparision with original data. The absolute error and percentual error will be displayed eather.
